# gep-fn-skyrunchallenge
**게임엔진실습 언리얼 기말과제**

**3D 플랫포머 게임** (**블루 프린트**로 구현)

**구현 목록**

1) 이동하는 바닥
2) 밟으면 일정 시간 후 물리력이 생기는 바닥
3) 회전하는 플랫폼 물체 3개 이상
4) 바닥으로 떨어지면 세이브 포인트 위치로 순간이동
5) 사운드 효과
6) 시작점과 결승점에 통과 시 UI 및 기본 나이아가라 효과
7) 게임 재시작
8) terrain 제작, 메테리얼 적용
9) 등등
